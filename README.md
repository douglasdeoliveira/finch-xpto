## :information_source: Como utilizar

```bash
# Faça o clone do repositório
git clone https://github.com/douglasdeoliveira/finch-xpto

# instale as dependências
cd finch-xpto && yarn

# frontend
yarn start

# test
yarn test

# storybook
yarn run storybook
```
